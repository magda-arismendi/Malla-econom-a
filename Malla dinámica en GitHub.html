🔧 ¿Qué necesitas?

  1. Cuenta de GitHub (ya debes tener una).

  2. Conocimientos básicos de Markdown y/o HTML/CSS/JS.

  3. Una idea clara de tu malla curricular (asignaturas, semestres, etc.).

📌 Opciones de formato

A) Versión básica con Markdown dinámico

Ideal para el README del repositorio. Puedes hacer "checklists" por semestre y marcar tu avance.

B) Versión web interactiva (HTML/CSS/JS)

Para que se vea como una página web con efectos visuales. Puedes usar tecnologías como React o simplemente HTML + JS.

🧠 Opción B: Página web dinámica

Si prefieres una malla interactiva, te puedo ayudar a crear una web con HTML/JS donde puedas marcar ramos aprobados. Aquí te dejo un ejemplo base:

Archivos necesarios:

📄 index.html

html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Malla Curricular - Ingeniería Comercial</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>📘 Malla Curricular</h1>
  <div id="malla"></div>
  <script src="script.js"></script>
</body>
</html>

🎨 style.css

css
body {
  font-family: Arial, sans-serif;
  padding: 20px;
}

.semestre {
  margin-bottom: 20px;
  padding: 10px;
  border: 1px solid #aaa;
  border-radius: 10px;
}

.semestre h2 {
  margin-top: 0;
}

.ramo {
  display: flex;
  align-items: center;
  margin: 5px 0;
}

.ramo input {
  margin-right: 10px;
}

🧠 script.js

javascript
const malla = {
  "1er Semestre": [
    "Introducción a la Economía",
    "Cálculo I",
    "Fundamentos de Administración",
    "Contabilidad",
    "Taller de Expresión Escrita"
  ],
  "2do Semestre": [
    "Microeconomía I",
    "Álgebra",
    "Derecho Comercial",
    "Estadística",
    "Ética y Sociedad"
  ]
};

const contenedor = document.getElementById("malla");

for (const [semestre, ramos] of Object.entries(malla)) {
  const div = document.createElement("div");
  div.className = "semestre";
  div.innerHTML = `<h2>${semestre}</h2>`;
  
  ramos.forEach(ramo => {
    const label = document.createElement("label");
    label.className = "ramo";
    label.innerHTML = `<input type="checkbox"> ${ramo}`;
    div.appendChild(label);
  });

  contenedor.appendChild(div);
}

🚀 Cómo publicarlo en GitHub

  1. Crea un repositorio nuevo en GitHub (ej. malla-curricular).

  2. Sube los tres archivos (index.html, style.css, script.js).

  3. Activa GitHub Pages desde la pestaña Settings > Pages → selecciona la rama main y la carpeta /root.

  4. Accede a tu sitio en https://tuusuario.github.io/malla-curricular.

  2. Sube los tres archivos: index.html, style.css, y script.js.

  3. Ve a Settings > Pages.

  4. En "Source", selecciona la rama main y carpeta /root.

  5. Guarda. En unos segundos, tu página estará en:

✅ PASO 1: Crea una cuenta en GitHub (si no tienes)

  1. Ve a https://github.com/

  2. Si ya tienes cuenta, inicia sesión.
    Si no, haz clic en "Sign up" y sigue las instrucciones para crear una.

✅ PASO 2: Crea un nuevo repositorio

  1. Una vez dentro, haz clic en el "+" arriba a la derecha y selecciona "New repository".

  2. Ponle un nombre, por ejemplo:
    👉 malla-economia

  3. Asegúrate de que esté en modo público.

  4. Marca la opción: ✅ "Add a README file"

  5. Haz clic en el botón "Create repository"

✅ PASO 3: Sube los archivos de tu malla

Ahora vamos a subir 3 archivos: index.html, style.css y script.js. Para eso:

A. Abre tu repositorio y haz clic en el botón verde que dice "Code" > "Add file" > "Upload files"

B. Aquí puedes hacer drag and drop (arrastrar) los archivos o hacer clic en "choose your files"

Voy a prepararte esos 3 archivos comprimidos en un .zip para que lo descargues, extraigas y los subas tú misma. Aquí está el link de descarga:

➡️ Descargar archivos de la malla (.zip)

C. Una vez que los subas, baja a la página y haz clic en el botón verde "Commit changes"

¡Ya tienes los archivos en tu repositorio!

✅ PASO 4: Activa GitHub Pages

  1. En tu repositorio, haz clic en la pestaña de "Settings"

  2. En el menú de la izquierda, busca y haz clic en "Pages"

  3. En la sección "Source", elige:

    • Branch: main

    • Folder: / (root)

  4. Haz clic en "Save"

📌 GitHub va a generar tu sitio en unos segundos. Te aparecerá un link como este:

arduino
https://tuusuario.github.io/malla-economia/
